阅读对象

民生银行为合作的移动收单商户提供了SDK接入的方式，本文档主要介绍合作机构的开发者如何进行SDK服务端的开发，配合SDK移动端完成移动收单功能。

该指导主要面向合作机构服务端的开发人员。

版本说明

当前版本 V1.0 添加支付查询、微信支付退款接口说明 赵琦

准备工作

机构在与民生银行完成线下协议签订后，由总行乐收银商务平台管理员在乐收银商务平台开立机构平台号，平台号唯一，机构真实进行交易的商户须在民生乐收银商务平台开通商户，开通商户后，会生成民生银行商户号，民生会为每一个机构平台发放用于订单加密加签和解密验签的的通信证书。证书由商户后台保存，用于完成与民生银行之间数据加密解密工作。

其中，通信协议采用SM2算法，商户订单信息采用机构私钥证书签名(cust0001.sm2)，民生公钥证书(cmbcTest.cer)加密。商户收到民生银行返回的信息，用商户私钥证书(cust0001.sm2)解密，民生公钥证书(cmbcTest.cer)验签，确保双方交互的信息加密传输，安全可靠防抵赖。

民生银行目前提供加密加签和解密验签demo，支持的语言有c#.net，php，和java，商户可以下载demo结合文档进行开发，测试证书在民生银行开发平台直接下载。

收单SDK开发需要与商户后台开发同时进行，SDK接入流程如下：

1.商户后台开发：根据本文指导，在商户后台集成民生通信证书，根据不同的功能API字段完成相关功能的请求字段签名、加密工作。

2.商户APP集成SDK：在商户原有APP基础上集成民生移动收单SDK，根据本指导，完成SDK各模块的集成开发工作。

3.商户APP调用SDK发起收款等功能测试工作：在APP配置文件中填写UAT测试环境地址，完成各模块的联调工作。

4.商户APP上线验收工作：在确保测试环境联调无误后，将APP配置文件中UAT环境参数改为生产环境参数，进行最后的生产环境下调试验证工作。

功能描述

民生移动收单插件，主要面向有收单需求的商户APP，实现移动端收单服务快速接入。商户需要收单时，产生一笔收单订单，调起民生收单插件，由民生收单插件完成订单支付流程，订单支付成功后，民生后台会异步通知商户后台，完成整个收单过程。

案例介绍

目前已上线支付案例，商户可进行实际体验。

商户案例：福州273、乐收银App、冠达星云商、临沂云物流等。

业务流程

商户系统集成SDK后下单主要交互说明：

步骤1：在商户APP中选择对应的服务或商品，发起下单申请给后台；

步骤2：商户后台将订单信息用本地通信密钥签名并加密，将密文传至SDK；

步骤3：收单SDK将加密订单信息发送至民生银行收单网关；

步骤4：收单SDK会发起和民生银行建立安全通道；

步骤5：银行端将解密后的订单信息在安全通道上返回；

步骤6：收单SDK发起下单请求；

步骤7：民生系统完成订单校验并完成下单支付操作；

步骤8：收单SDK发起支付查询结果；

步骤9：收单SDK收到支付结果后在商户App端显示支付结果；

步骤10：异步通知商户后台支付结果。

订单生成

民生会为每一个接入平台分配一个平台号，民生会为平台发放订单生成的证书。对于接入的商户，民生会为商户分配一个商户号。一个平台可以对应多个商户。交易时，由商户服务端按照民生要求使用民生提供的证书对订单内容进行加密加签，生成订单密文，将订单密文放到orderinfo字段中，并发送给App端SDK处理。

商户需开发实现订单结果接收接口，民生会向商户的notifyUrl中发送POST请求：内容为context=context密文，其中context密文解密解签后字段。

支付结果查询

民生提供给商户的微信支付结果查询接口，可查询刷卡，微信正扫、反扫、以及退款交易的结果，只可查询当日交易。

请求地址：
测试环境 http://wxpay.cmbc.com.cn/mobilePlatform/appserver/paymentResultSelect.do

加解密说明

民生提供了对密文加密加签、解密解签的jar包。

加密加签使用
EncryptAndSignService custEncrypt = new EncryptAndSignService(reqBodyJson);
if(custEncrypt.custEncryptAndSign(priKeyAbsPath, certAbsPath, priKeyPWD)){
	cipherTxt = custEncrypt.getCipherTxt();
}
加密加签参数
reqBodyJson 需加密的明文
priKeyAbsPath 商户私钥绝对路径
certAbsPath 民生公钥绝对路径
priKeyPWD 商户私钥密码
解密解签使用
DecryptAndVerifyService custDecrypt = new DecryptAndVerifyService(cipherTxt);
if(custDecrypt.custDecrypyAndVerify(priKeyAbsPath,certAbsPath, priKeyPWD)){
	plainBody = custDecrypt.getPlainBody();
}
解密解签参数
cipherTxt 需加密的明文
priKeyAbsPath 商户私钥绝对路径
certAbsPath 民生公钥绝对路径
priKeyPWD 商户私钥密码
